原本telus采用120模组当时测试出一些异常，与NB组沟通后他们建议更换模组
目前那边测试通过的是使用150模组的板子。
若要生产请确认NB模组和它的版本为最新


1.源码中屏蔽了整机的出厂检测
若要生产需修改屏蔽如下语句：
修改前：
#warning 调试用，屏蔽出厂检测!!   
   PT_flag[0] = SET;

修改后：
#warning 调试用，屏蔽出厂检测!!   
//   PT_flag[0] = SET;

2.杨彬这边之前硬件小批的时候LDO错买成了3.3V的，后续若生产时硬件改回来了
软件需修改如下地方：
修改前：
AD_Value  = (AD_Value*3200) >> 12;

修改后：
AD_Value  = (AD_Value*3000) >> 12;
